# Guess The Number Game

## What is it?
A pixel-art styled guessing game where players try to guess a secret number between 0-50 in 10 attempts.

## How to Play
1. Enter a number (0-50) in the input box
2. Click "Check Guess" to submit
3. The mystery box rotates and shows if you're correct
4. You have 10 attempts to guess the right number
5. Click "New Game" to play again

## Features
- Input validation (numbers only, max 2 digits)
- Real-time attempt tracking
- Animated mystery box with game feedback
- Win/Loss screen with the correct number revealed
- Reset button to clear current guess
- Game locks after win or loss

## Controls
- **Check Guess**: Submit your guess
- **Reset**: Clear the input field
- **New Game**: Start a fresh game

## Files
- `index.html` - Game structure and styling
- `script.js` - Game logic and functionality

## Browser
Works on all modern browsers (Chrome, Firefox, Safari, Edge)
